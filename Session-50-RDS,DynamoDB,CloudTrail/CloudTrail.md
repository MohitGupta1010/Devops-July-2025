# AWS Cloud Trail

- records every API call and Actions made in your account
- like who did, what, when, where from which region you can access

## How to Implement the same

- Aws console - cloudtrail
- create trail
- name: my-security-trail
- Management Events - check on that
- create S3 bucket for storing logs
- you can also select clud watch
- continue with default options and create trail

## Once Its done 

- try to crete instance
- stop instance
- terminate instance

- open trail and check event history

- after 6-10 minutes check s3 bucket there also you can see
- region wise folders and select region and date and month wise folders logs you can see.
